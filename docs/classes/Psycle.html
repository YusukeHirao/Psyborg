Parameters:

  • $el JQuery

    インスタンス化する要素

  • options Any
    • [instanceKey='psycle'] String optional

      dataメソッドで取得できるインスタンスのキー文字列

    • [startIndex=0] Number optional

      最初に表示するパネル番号

    • [transition='slide'] String optional

      トランジションの種類

    • [duration=600] Number optional

      アニメーション時間

    • [easing='swing'] String optional

      トランジションのイージング

    • [delay=3000] Number optional

      オートプレイの時の待機時間

    • [auto=true] Boolean optional

      オートプレイかどうか

    • [cancel=true] Boolean optional

      アニメーション中にキャンセル可能かどうか（アニメーション中にパネル選択やパネル送りを上書きできるかどうか）

    • [repeat=PsycleRepeat.RETURN] Any optional

      繰り返しの種類(NONE: 繰り返ししない, RETURN: 最後まで到達すると最初に戻る, LOOP: ループしてるかのように最初に戻る（ループに対応しているトランジションのみ））

    • [container='>ul:eq(0)'] String optional

      コンテナを取得するためのセレクタ

    • [panels='>li'] String optional

      パネルを取得するためのセレクタ（コンテナからのパス）

    • [currentClass='current'] String optional

      現在のパネルに設定されるクラス名

    • [delayWhenFire=0] String optional

      遷移処理が発生する(before関数からfire関数)までの遅延時間(ミリ秒)

    • [clone=2] Number optional

      ループリピートにしたときの各要素に対してのクローン要素の数

    • [cols=1] Number optional

      カラム(列)の数（カラム対応のトランジションのみ）

    • [rows=1] Number optional

      行の数（行対応のトランジションのみ）

    • [offsetX=0] Number optional

      コンテナの横方向のオフセット（コンテナが平行移動するトランジションのみ）

    • [offsetY=0] Number optional

      コンテナの縦方向のオフセット（コンテナが平行移動するトランジションのみ）

    • [nearby=false] Boolean optional

      ???

    • [innerFocus=false] Boolean optional

      マルチカラムの時のフォーカスの当たり方が内側優先かどうか、noFocusがtrueの場合は無効

    • [noFocus=true] Boolean optional

      マルチカラムの時、パネルにフォーカスを当てない、また、indexは先頭の要素だけを指すことになる

    • [resizable=false] Boolean optional

      リサイズによってパネルの大きさが変わる場合はtrueを渡す

    • [draggable=false] Boolean optional

      ドラッグによって遷移をコントロールさせる場合はtrueを渡す

    • [swipeable=false] Boolean optional

      スワイプによって遷移をコントロールさせる場合はtrueを渡す

    • [dragBlockVertical=false] Boolean optional

      ドラッグの上下を抑制させる(タッチデバイスのスクロールも抑制される)場合はtrueを渡す

    • [bindKeyboard=false] Boolean optional

      キーボードで操作できるようにするかどうか

    • [showOnlyOnce=null] Any optional

      オートプレイの時に一度しか表示しないパネルのフィルタセレクタ (例) .once

    • [controller=null] Any optional

      コントローラ

    • [marker=null] Any optional

      マーカー

    • [thumbnail=null] Any optional

      サムネイル

    • [css3=true] Boolean optional

      ???

    • [loopCloneLength=null] Number optional

      ループ時のスライド専用 クローンをいくつつくるか

    • [scenes=[] Function optional

      ???
